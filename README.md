# Arya & Jainam Cinematic Wedding Website

A from-scratch Next.js wedding website with a cinematic, warm, intimate Malibu scroll experience.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Notes

- The RSVP is currently a front-end demo interaction. Connect it to Google Forms, Airtable, Supabase, or another backend when ready.
- Photos are in `public/photos`.
- Main page content is in `app/page.tsx`.
- Styling and animations are in `app/globals.css`.
